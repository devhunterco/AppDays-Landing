Inscribete yá!

Mozilla Colomia y DevHunter los invita a participar en el primer App Day Monteria que se realizara los dias del 24 al 26 de Abril en el Centro de Convenciones de Montería unas sesión sobre Firefox OS, el sistema operativo libre para la web movil de Mozilla.

Si eres desarrollador web, podrás desarrollar aplicaciones en HTML5, JavaScript y CSS para Firefox OS. Te ayudaremos con todo lo demás, incluyendo el acceso a las APIs web y cómo usar Firefox OS Simulator en tu navegador de escritorio para ver y probar tus aplicaciones móviles.

Técnicos de Mozilla compartirán herramientas y la tecnología para ampliar y apoyar esta nueva plataforma, y estarán disponibles para ayudar a los hackers.

Este "App Day" es una oportunidad para iniciarse en la creación de aplicaciones móviles para el Firefox Marketplace y el próximo dispositivo Firefox OS. También es una gran ocasión para mostrar tu proyecto a una audiencia de compañeros, así como a líderes tecnológicos e innovadores.

Qué Traer!

Trae tu máquina de desarrollo Linux, Mac o Windows, Tablets u otros dispositivos que utilices para plasmar tu idea. Nosotros te proporcionaremos todo lo demás.

Qué crear!

Apps. El Firefox Marketplace busca aplicaciones web abiertas que funcionen bien en dispositivos móviles para Firefox OS y Firefox para Android. Pero estamos interesados en todo tipo de aplicaciones. Puedes traer una aplicación HTML5 que has iniciado para continuarla, finalizarla,pulirla o simplemente presentarla.

Ven y desarrolla tus ideas usando herramientas OpenWeb para que tu aplicacion se pueda ver en diferentes dispositivos como Laptops, Tv's, Tablets, Etc.
